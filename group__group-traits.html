Whether T can be converted to Variant.

The intrusive way to enable the trait for user defined type T is to implement Variant toVariant(T const&) static member for T.

The non-intrusive way to enable the trait for user defined type T is to provide specialization

namespace yenxo {
template <>
struct ToVariantImpl<T> {
Variant apply(T const&) {
// implementation
}
};
}
