Detailed Description

template<typename T>
class yenxo::Pimpl< T >

Generic 'Pimpl' idiom implementation.

Member functions not marked noexcept do throw if and only if T or allocation throws. Respects const correctness.

Example

#pragma once
#include <yenxo/pimpl.hpp>
class X {
public:
X();
~X();
X(X const&);
X& operator=(X const&);
X(X&&);
X& operator=(X&&);
private:
struct Impl;
yenxo::Pimpl<Impl> impl;
};
